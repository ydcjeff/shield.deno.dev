# [shield.deno.dev](https://shield.deno.dev)

> Badge service for Deno modules

Replace https://deno.land with https://shield.deno.dev in the deno module URL:

```diff
- https://deno.land/x/oak
+ https://shield.deno.dev/x/oak
```
